# HR Analytics: Employee Attrition Analysis

This repository contains a data analytics case study on employee attrition using the IBM HR Analytics Employee Attrition & Performance dataset.

The project examines employee turnover patterns and identifies factors associated with attrition, including overtime, income, job satisfaction, work-life balance, department, job role, age, experience, and tenure.
